## Repackaging
```
tar xzf vim.tar.gz
(... doing something...)
tar czf vim.tar.gz .vimrc .vim .bashrc_prompt
```

## Build

```
./build.sh

```

## Install

```
wget https://github.com/5bpa/v/raw/master/v.sh
bash v.sh
```
